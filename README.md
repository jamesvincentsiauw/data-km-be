# DATA KM ITB BACKEND SERVICES

## Technology
- NodeJS
- CouchDB
- GraphQL

## How to Use
```bash
# Clone Repository
$ git clone https://github.com/jamesvincentsiauw/data-km-be.git

# Install Dependencies
$ npm install

# Create .env File
$ cp .env.example .env

# Modify .env File with Your Own Configuration

# Run the Script
$ npm start

# GraphQL API started at http://localhost:4000/
```
